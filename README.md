Regularly translated investigative findings into scalable detection logic and platform-level protections. Leveraged clustering and pattern correlation across accounts, devices, and transaction flows to identify coordinated activity.

### AI-Driven ATO Detection Model (Experimental Development)

**Developed and trained an AI-based detection model** focused on account risk scoring across multiple business dimensions (Customer, Merchant, and Driver ecosystems). Created custom labeling frameworks and feature weighting systems to reflect real-world fraud patterns and business impact.

Iteratively tuned model performance by adjusting feature weights, balancing detection sensitivity with false positive rates, and validating outputs against known fraud cases. Special attention was given to data quality and minimizing model hallucination through controlled training inputs and validation loops.

Utilized supervised learning with custom-labeled datasets and feature weighting to optimize precision/recall trade-offs.

This project represented a shift toward more adaptive, scalable detection systems beyond static rules-based approaches, enabling more resilient detection against evolving attacker behavior.
